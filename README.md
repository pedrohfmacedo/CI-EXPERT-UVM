# Atividade 1
## Descrição

Este repositório contém a implementação da **Atividade 1 do CI-Expert**, baseada no material apresentado na Aula 1.

O objetivo principal é estruturar um projeto de hardware digital utilizando boas práticas de organização, automação e validação, incluindo:

- Script de geração de projeto (`gen-prj.sh`)
- Código RTL
- Testbench (tb)
- Estrutura organizada de diretórios
- Execução funcional e reproduzível
